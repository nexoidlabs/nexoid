# NX-Platform — Web Application

Full-featured web management dashboard. The complete Nexoid experience.

Answers: **"How do I manage everything in the Nexoid ecosystem?"**

## Key Components

- **Identity & Credentials** — Full credential lifecycle management
- **Wallet & Assets** — Safe wallet management, multi-wallet view
- **Chat** — Conversational interface with agents
- **Mission Control** — Kanban-style task board for agent task assignment
- **Fleet Management** — Agent status, health, spending, delegation overview
- **Delegation Center** — Visual delegation tree with scope configuration
- **FinOps Dashboard** — Real-time spend tracking across the agent fleet
- **Audit Log** — Structured event log, filterable by agent and time

## Relationship

Pure consumer — reads from and writes to other modules. All mutations flow through NX-Control Plane.
